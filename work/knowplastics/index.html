kNOw Plastics

Educational multiplayer online game

kNOwPLASTICS Online Game is a multi-player memory game based on Memory, also known as Concentration, Pelmanism, Shinkei-suijaku, or Pairs. The objective is to find matching pairs of cards. We developed both the game engine and the in-browser app.

WasteLess India is a non-profit social enterprise based in Auroville, South India. Their mission is to transform the way we think about waste by promoting responsible waste management and environmental sustainability. They developed an educational program, kNOwPLASTICS, which includes a memory game. Teachers can use the multi-week program to educate young students (6-12 years) on the importance of reducing the use of unsafe and single-use plastics.

Play the game now: knowplastics.servegame.com

  • Production: WasteLess India
  • Design: Impprintz
  • Illustrations: Navleen Kohli
  • Programming: Office 4 (John Zandbergen)

Project

Our objective was to utilize the illustrations, design, and descriptions of game stages to create a turn-based, multi-user online game. We used Angular for the client, NodeJS for the game server, and hosted the project on Heroku.

Most notable requirements

  • Multi-User: Children choose a game-group and wait until others join the game.
  • No registration is required.
  • Game Server: Regulates game progress, giving users 15 seconds for their turn.
  • Responsive Design: Must work on different screen sizes.

Technology

HTML5, CSS3, CSS Animations, Angular, NodeJS, TypeScript, Javascript, HAPI, MongoDB
